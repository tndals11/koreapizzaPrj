관리자페이지

홈화면 
- 회원 수, 총 매출, 고객문의, 고객문의 답변완료, 현재 리뷰평점, 누적 주문횟수 구현
- 홈화면에 주문목록, 상품목록, 회원목록, 문의목록 화면에 띄우기

주문관리
- 사용자가 주문을하여 결제를 하면 주문관리 페이지에 등록이 됩니다.
- 주문취소 구현 

메뉴관리
- 관리자 상품등록 페이지에서 상품을 등록하면 사용자 메뉴페이지에 상품의 이름, 가격, 코멘트가 등록됩니다. 
- 상품 등록시 관리자 메뉴페이지 리스트에 등록이 되고 아이템 보기,수정,삭제 구현을 했습니다.

회원관리
- 사용자가 회원가입시 관리자 회원목록 리스트에 저장됩니다.
- 사용자가 회원탈퇴시 회원상태에 탈퇴라고 표시가 됩니다.
- 사용자 상세보기,삭제 구현

리뷰관리
- 사용자가 주문 후 리뷰를 작성하면 리뷰리스트에 등록됩니다.

문의관리
- 사용자가 1:1문의 시 관리자 문의관리 리스트에 등록됩니다.
- 사용자의 1:1문의를 보고 관리자가 답변을 달수있게 구현했습니다.
- 관리자가 답변시 제목 옆에 [답변완료]라는 코멘트를 달게 구현했습니다.

사용자페이지 

회원가입 
- 회원가입 페이지에서 회원가입시 유효성검사와 아이디 중복체크를 구현하였습니다.

로그인
- 사용자가 활성화 상태일시는 메인화면으로 
- 사용자가 회원탈퇴 상태일시에는 "이미 탈퇴한 회원입니다"라고 띄웁니다
- 특정한 아이디와, 비밀번호를 입력시 관리자 메인페이지로 이동합니다.
- 아이디 찾기, 비밀번호 찾기 구현

메인화면 
- 비로그안시 메인페이지와 로그인시 메인페이지가 다르게 구현
- 비로그인 상태에서 리뷰쓰기나, 주문내역, 장바구니를 클릭할 시 "로그인 후 이용해주세요" 라는 메세지를 보여줍니다.

장바구니
- 장바구니에 아무것도 없을시, 비어있는 장바구니 페이지로 이동
- 장바구니에 아이템이 존재할 시 결제페이지로 이동

메뉴페이지
- 관리자 페이지에서 등록한 상품을 사용자에게 보여줍니다
- 사용자가 장바구니에 담을 시 하단 상태바에 결제금액과 아이템수량이 변경되도록 구현하였습니다.
- 하단 상태바에 장바구니 클릭시 결제페이지로 넘어갑니다.

결제페이지
- 메뉴를 각각 삭제 할 수 있는 기능과 전체삭제 할 수있는 기능을 구현하였습니다.
- 사용자 요청사항 구현
- 사용자가 결제시 카카오결제 API를 구현하였습니다.

주문내역
- 사용자가 결제를 끝내면 주문내역 리스트에 등록됩니다.
- 사용자가 리뷰 작성을 안했을 시 리뷰쓰기 버튼이 활성화됩니다.
- 사용자가 리뷰 작성을 완료했을시 작성완료 버튼으로 변경되며 버튼이 작동하지 않습니다.

내정보 
- 주문내역, 1:1문의, 내 정보 수정 , 비밀번호 변경, 문의 게시판 구현

1:1문의
- 1:1문의를 클릭시 회원정보를 보여주며 문의유형을 선택한 후 문의를 작성 할 수 있습니다.

1:1문의 게시판
- 시용자가 등록한 1:1문의를 보여주고 관리자가 답변을 달시 사용자 답변한 코멘트가 달립니다.

내 정보 수정 
- 이메일을 통해 본인확인 후 회원정보를 수정 헐 수 있습니다.
- 내 정보보기, 정보수정, 회원탈퇴 구현

비밀번호 변경
- 비밀번호 변경 구현
